## Web-версия заданий JS.

При первом запуске пользователю отображается перечень тем на выбор. Для дальнейшего использования сайта надо открыть консоль, это можно сделать, нажав правую кнопку мыши, затем во всплывающем меню нажать просмотреть код, потом выбрать вкладку «Console». При клике на кнопку «6 The Secret Life of Objects» на странице и в консоли отобразятся описания всех заданий в этой теме, так же в консоли отобразится результат выполненного кода, Исходный код можно посмотреть во вкладке «Sources».
